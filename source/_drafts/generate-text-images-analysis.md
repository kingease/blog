# 合成图像

主要是生成字符的mask, 然后用mask将背景和颜色混合在一起。


## 生成mask

几种生成需求
1. 指定长度的text, 每一个char是字符集中随机选取
2. 有些字符，在某些字体中并不存在
3. 特定字符的指定生成
4. 特定text pattern的指定，比如日期、编码、号码
5. 生成的mask的长度要固定，生成的字符最好不能超出范围
6. 将来不排除生成缺失性的字符


综合上述的需要
1. 字体
2. 生成的字符串
3. 配置
